# 📜 SmartFinance: The Gothic Ledger

[![Expo SDK 55](https://img.shields.io/badge/Expo-SDK%2055-blue.svg)](https://expo.dev/)
[![Design System](https://img.shields.io/badge/Theme-Gothic%20Ledger-indigo.svg)](https://fonts.google.com/specimen/Grenze+Gotisch)

SmartFinance is a premium, high-contrast financial management application for mobile. Transformed from a modern fintech aesthetic into a specialized **Medieval Gothic Ledger**, it combines historical elegance with cutting-edge financial utility.

---

## 🌌 The Vision
SmartFinance is more than an expense tracker; it is a **Financial Matrix**. Inspired by ancient ledger books and modern glassmorphism, the interface provides a "Neural Link" to your wealth. Every interaction is designed to feel premium, tactile, and secure.

## 🎨 Design System: Medieval Gothic
- **Typography (Grenze Gotisch)**: Primary headings and titles evoke the authority of a historical script.
- **Typography (Cinzel)**: Transactional data and UI labels maintain readability with a classic stone-carved aesthetic.
- **Surfaces**: Deep space backgrounds with translucent **GlassCards**, vibrant indigo glows, and fluid **Moti** animations.

---

## 🏛 Core Modules

### 1. Main Console (Dashboard)
- **Net Asset Valuation**: Real-time tracking of total capital.
- **Yield Delta Scanner**: Projected growth analysis with interactive modeling.
- **Health Score**: A proprietary algorithm measuring your fiscal stability.

### 2. Protocol Ledger (Expenses)
- **Spectral Entry**: Quick-tap transaction logging with smart categorization.
- **Fiscal Distribution**: Visual breakdown of outflows by sector (Groceries, Tech, Bullion, etc.).

### 3. Wealth Matrix (Investments)
- **Bullion Tracking**: Monitor gold and silver holdings.
- **Capital Influx**: Manage SIPs and mutual fund units with ease.

### 4. Financial Protocols (Schemes)
- **AI-Driven Logic**: Personalized recommendations for government and banking schemes.
- **Eligibility Validator**: Instantly check if your profile meets protocol access requirements.

## 🛠 Technology Matrix

- **Framework**: Expo (React Native) SDK 55
- **Styling**: NativeWind (Tailwind Architecture)
- **Animations**: Moti & Reanimated 4
- **Backend**: Firebase Firestore & Authentication
- **Icons**: Lucide React Native (Stroke Width 2.5)

## 🚀 Deployment Guide

### Prerequisites
- [Node.js](https://nodejs.org/) (v18+)
- [Expo Go](https://expo.dev/go) on your mobile device

### Installation
1. Clone the repository:
   ```bash
   git clone git@github.com:nilacandassamy-29/smartfinance.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Initialize the matrix:
   ```bash
   npx expo start
   ```
